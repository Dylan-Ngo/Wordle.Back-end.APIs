## Run and test the services:

1. Initialize the databases:

    ```
    ./bin/init.sh
    ```

2. Start the services:
    ```
    foreman start
    ```

3. Test the API calls using curl, HTTPie, or 
   `http://localhost:5000/docs` and `http://localhost:5100/docs`
